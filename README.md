### `$ whoami`

Fourth-year Computer Science student at Universidad Nacional del Sur (Argentina), focused on systems programming: distributed systems, networking, and infrastructure. Currently open to remote internships and part-time roles.

Everything I use runs on my own hardware: the portfolio behind that badge is served from a Raspberry Pi through Caddy, WireGuard, and nftables. Self-hosting is the fastest way I know to learn how things actually work, and when something breaks at 3 AM, at least it's my fault.

### `$ ls ~/projects/`

| Project | What it is | Built with |
|---|---|---|
| [godfs](https://github.com/FrancoJ-cmd/godfs) | Fault-tolerant distributed file system: heartbeat failure detection, block replication, replica fallback on reads. Standard library only. | `Go` |
| [HitList](https://github.com/FrancoJ-cmd/HitList---Gaming-Hub) | Desktop app ranking Steam's most played games in real time: live player counts + review ratios, offline-first caching, Clean Architecture. | `Kotlin` `Compose` |
| [tfidf-summarizer](https://github.com/FrancoJ-cmd/tfidf-summarizer) | Extractive summarizer for Spanish text: Rust pipeline with a spaCy NER sidecar over a JSON stdin/stdout protocol. | `Rust` `Python` |
| [ketobytes](https://github.com/FrancoJ-cmd/ketobytes) | Bakery management CLI with recipe-based pricing: update one ingredient and everything is repriced. | `Python` `SQLite` |
